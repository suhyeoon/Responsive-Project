메인컨텐츠

창호예술을 선도하는 기업 70년 역사의 기술과 명성, 남선알미늄이 만들면 다릅니다.
자연과 인간과 기술의 조화 자연과 인간과 기술이 어우러진 21세기 글로벌 컴퍼니, 남선알미늄
품위있는 공간 기쁨있는 생활 협동과 창조와 도전정신을 갖춘 남선인으로서 역량을 발휘합니다.
1991년 설립된 기술연구소 기술혁신을 통해 세계적인 기업으로 도약하기 위해 최선을 다할것입니다.

홍보영상

“품위있는 공간 기쁨있는 생활” 남선알미늄의 홍보영상 입니다.

주요 컨텐츠 바로가기

  • 남산역사관

    남선알미늄의 역사관 입니다.

  • 사업영역

    생산, 판매하고 있습니다.

    자세히 보기 +

  • 인재채용

    자세히 보기 +

제품소개

최고급 시스템창을 중심으로 하는 알루미늄 창호뿐만 아니라 PVC창호사업까지, 남선알미늄의 제품을 소개합니다.

  • 창호

    윈다트 / 가안샤시(PVC) / 뉴알플러스(복합창)

  • 도어

    시스템도어 / 가안도어

  • 커튼월

    알플러스 타입 / 폴리아미드 단열 타입 / 아존+폴리아미드 단열 타입 / 아존 타입 / 기타

  • 산업구조재

    산업구조재

  • 남선방폭창

    SLIDING / FIX / TILT&TURN / PROJECT / CASEMENT / CURTAIN WALL / DOOR

혁신기술

70년 남선을, 미래의 남선을 있게 할 핵심기술은 품질입니다.
  • 01

    연구소 소개

    21세기 무한경쟁시대에 필요한
    신기술 개발과 신제품 개발에
    최대한의 노력을 경주합니다.

  • 02

    개발분야 및 실적

    알루미늄 창호뿐만 아니라
    PVC창호사업을 통해 명실상부한
    창호전문회사로 성장하고 있습니다.

  • 03

    사이버 공정도

    원자재의 사용부터 가공까지
    남선알미늄의 특별한
    사이버 공정도를 확인하세요.

  • 04

    남선알미늄 특허

    기술개발과 경영혁신을 통해
    이룩해 놓은 기반을 바탕으로
    국내 산업발전을 선도하겠습니다.
